## Cài đặt

```bash
# Clone repository
git clone https://github.com/EGANY-Team/vct-14-cypress
cd vct-14-cypress

# Install package
npm install

# Start dev server at http://localhost:3000
npm start

# Run cypress
npm run cypress

# Build for production
npm run build
```
